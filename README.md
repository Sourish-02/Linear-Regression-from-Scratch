# Linear-Regression-from-Scratch
Aim: Implementation of Linear Regression from scratch using the Normal Equation, Batch Gradient Descent, SGD, and L1/L2 regularisation with experimental comparison.

Test and Train data used from https://www.kaggle.com/datasets/andonians/random-linear-regression?resource=download

For the training and test data used,
1. Closed Form (Normal Equation)
    MSE: 9.434852832251472
    R2 score: 0.9887991524196075
